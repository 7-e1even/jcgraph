# jcgraph source/sink catalog for taint-style vulnerability hunting.
#
# Format (whitespace-separated, 3 columns):  <kind> <category> <owner>#<name>
#   kind     = sink | source
#   category = free-form tag used for grouping/filtering
#   owner    = JVM internal name (slashes); name = method name (no descriptor)
#
# Matching is exact on method name (owner#name followed by '(' in the call target),
# so all overloads of a name match but sibling names do not. The owner is the
# DECLARED type at the call site (what ASM records), which is exactly how the
# call graph stores CALLS targets — interface owners like HttpServletRequest match.
#
# Lines starting with '#' and blank lines are ignored. Edit freely.
